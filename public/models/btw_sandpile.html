BTW Sandpile

Bak–Tang–Wiesenfeld self-organized criticality · perspective 3D view · steppable grain → sweep → avalanche · frozen-config A/S probes

Run

Slowest → fastest: a single toppling, one sweep, one whole avalanche, … up to one grain added per site (a full lattice pass) each frame.
L snaps to sizes giving a whole number of pixels per lattice site. Below L = 100 the pile is 3-D balls; above 100 it is columns — flat squares when viewed top-down (tilt 0°), real 3-D columns once you tilt.
Drive mode (grain drop site)
Det. boundary drives the top boundary row one site at a time, left → right, then repeats.
Frozen-config A / S
running
Freeze pauses the run and snapshots the configuration; press again to restart. Compute A & S then probes the frozen state: drop one grain at every site in turn (resetting each time). S[p] = size of the avalanche from p. A = total topplings at each site summed over all probe-avalanches.
Colour scheme (all panels)
Applies to the pile, A, S and the latest-avalanche map. Each panel is normalised to its own maximum.
Pile view
Force the flat top-down colour map (no 3-D) at any size — handy as a fast view for very large L when tilted 3-D gets heavy.
View (all 3 lattice figures)
Camera height for the A/S plots, in multiples of the tallest bar (its max value = 1): ×1, 2, 4, 8, 16, then ∞ (orthographic). Drag inside any lattice panel to tilt & turn them together.
