Invasion network

active latent
Arrow a→b = species a can invade b. Red = the species are neighbours on the grid now (invasion can fire); white = latent. Active only hides latent links and lays out just the live network. Palette: warm autumn reds, oranges, yellows, browns and olive greens. Each introduction adds two species at distant sites; with introduction rate = 0 a pair is added only when the lattice is frozen. The strip below the grid plots current diversity against cumulative added species; the y-axis spans 0 to the historical maximum, and patch count is rescaled so its peak meets the diversity peak.
